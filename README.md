## 🔐 Install locally

1. Clone the repo
   ```sh
   git clone https://github.com/lukaszzm/positivus.git
   ```
2. Enter to directory
   ```sh
   cd positivus
   ```
3. Install required packages
   ```js
   npm install
   ```
4. Run locally
   ```js
   npm run dev
   ```
